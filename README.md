# Swiggy Sales Analysis in Python

## 📊 Project Overview

This project analyzes Swiggy sales data using Python. The analysis focuses on 
identifying revenue trends,top-performing cities, and customer rating patterns 
using data visualization and exploratory data analysis techniques.

The objective is to demonstrate **data analysis skills using Python, Pandas, and Matplotlib**.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* seabort
* plotly
* Jupyter Notebook
